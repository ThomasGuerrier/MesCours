# Cours HTML5 et CSS3

Un dépôt de ressources et d'exemples de cours pour apprendre le HTML5 et le CSS3.

---

## Français

### Description

Cours HTML5 et CSS3 contient des ressources, des exemples et des exercices pour apprendre le HTML5 et le CSS3.

### Installation

1. Cloner ce dépôt ou télécharger les fichiers.
2. Ouvrir le fichier `index.html` dans un navigateur web.

### Utilisation

- Le point d'entrée principal est `index.html`.
- Tu peux naviguer dans les exemples et les ressources incluses.

### Contenu du dépôt

- `index.html` : page principale du projet.
- `style.css` : styles du projet.
- `Source-codes-CSS/` : dossier contenant des exemples et des exercices CSS.
- `fonts/` : polices utilisées par le projet.
- `images/` : images utilisées dans le projet.

### Contribution

N'hésite pas à proposer des améliorations ou à ajouter des exemples.

### Contact

- GitHub : https://github.com/ThomasGuerrier
